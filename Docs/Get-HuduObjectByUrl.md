# Get-HuduObjectByUrl

## SYNOPSIS
Get Hudu object from URL

## SYNTAX

```
Get-HuduObjectByUrl [[-Url] <Uri>] [<CommonParameters>]
```

## DESCRIPTION
Calls Hudu API to retrieve object based on URL string

## EXAMPLES

### EXAMPLE 1
```
Get-HuduObject -Url https://your-hudu-server/a/some-asset-1z8z7a
```
