# Remove-HuduAsset

## SYNOPSIS
Delete an Asset

## SYNTAX

```
Remove-HuduAsset [-Id] <Int32> [-CompanyId] <Int32> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Uses Hudu API to remove an Asset from a company

## EXAMPLES

### EXAMPLE 1
```
Remove-HuduAsset -CompanyId 1 -Id 1
```
