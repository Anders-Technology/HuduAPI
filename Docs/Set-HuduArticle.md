# Set-HuduArticle

## SYNOPSIS

## SYNTAX

```

 [[-CompanyId] <Int32>] [-ArticleId] <Int32> [[-Slug] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Uses Hudu API to update KB Article
